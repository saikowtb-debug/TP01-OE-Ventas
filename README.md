Archivo CSV con registros diarios de ventas (id, fecha, producto, cantidad, precio).

- /resultados: gráfico generado por el análisis

## Cómo ejecutar
1. Abrir Google Colab
2. Clonar el repositorio
3. Ejecutar: python scripts/analisis_ventas.py
